# トラック仕様書: プロジェクト基盤 & トップページ作成

## 1. 目的
プロジェクトの技術的土台を確立し、サイトの顔となるトップページを高品質なビジュアルとアニメーションで構築する。

## 2. 対象範囲
- Next.js (TypeScript, Tailwind CSS) の環境構築
- ダークモードを基調としたグローバルスタイルの設定
- 3Dグラフィックス (React Three Fiber) を用いたヒーローエリア
- スクロール連動アニメーション (Framer Motion) の導入
- お知らせセクション (ダミーデータ) の実装

## 3. 非機能要件
- Core Web Vitals (LCP, FID, CLS) の最適化
- モバイルフレンドリーなレスポンシブデザイン
