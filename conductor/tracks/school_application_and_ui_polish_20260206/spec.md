# トラック仕様書: スクール申し込みシステム構築 ＆ UI/UX極限ブラッシュアップ

## 概要
「生成AIブートキャンプ」の本番運用に向けた専用申し込みシステムの構築と、サイト全体のデザイン性を「明るい夜」および「夕焼けテクノロジー」をテーマに極限まで高める。

## 目標
- [x] スクール専用の申し込みページ（/school-application）の完全実装。
- [x] 決済手段（PayPal直接リンク、住信SBIネット銀行）の統合。
- [x] スクール専用のGAS連携（スプレッドシート転送、完全一致返信メール）。
- [x] サービスカードのビジュアル刷新（スポットライト、技術タグ、画像統合）。
- [x] トップページおよび詳細ページの不要な経歴情報の削除とクリーニング。

## デザインコンセプト
- **サイト全体**: 「明るい夜（Bright Night）」 - 深い闇の中でのテクノロジーの発光。
- **申し込みページ**: 「夕焼けの中のテクノロジー（Sunset Dawn）」 - 情熱的なオレンジとネイビーの融合。

## 主要コンポーネントの更新
### 1. サービスカード (ServiceCard.tsx)
- マウス追従のスポットライトエフェクト（シアン色）。
- 各サービス固有のヘッダー画像と技術タグ。
- 立体感のあるアイコン配置レイアウト。

### 2. スクール申し込みフォーム (/school-application)
- 夕焼けテーマの背景グラデーション。
- プルダウン主体の低負荷な入力体験。
- 支払い方法（PayPal/銀行）に応じた動的な完了画面。

## 技術スタック
- React / Next.js (App Router)
- Framer Motion (アニメーション)
- Lucide React (アイコン)
- Google Apps Script (バックエンド)
- PayPal (決済)
